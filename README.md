# DevBoard

A full-stack MERN project management application with Kanban boards.

## Features

- User authentication (JWT + cookies)
- Projects and tasks management
- Kanban board UI
- Infinite scroll and pagination
- Debounced search
- Real-time updates (Socket.io)
- Redis caching (planned)
- File uploads with AWS S3 (planned)

## Tech Stack

- React
- Node.js
- Express
- MongoDB
- Socket.io
- Redis
- Docker

npm install
npm run dev
